showQ :: Show a => Q a -> Q Exp #

Show string representation of template haskell expression, such as AST: $(showQ [| x -> x |])

This is useful to see how the code is represented

Show string representation of template haskell expression without Q Monad, such as names: $(showQ' 'id)
